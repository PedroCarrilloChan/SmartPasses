🚀 Primeros Pasos

Bienvenido a la documentación oficial de la API de Lealtad de Smart Passes. Esta API te permite gestionar mediante programación todos los aspectos de tus programas de lealtad para billeteras móviles.

🌐 URL Base

Todas las URLs a las que se hace referencia en esta documentación tienen la siguiente URL base:

https://pass.smartpasses.io/api/v1/loyalty/

🔐 Autenticación

Todas las peticiones a la API deben ser autorizadas con una App Key válida. La clave debe ser enviada en la cabecera Authorization de cada petición.

Ejemplo de Cabecera:

Authorization: TU_APP_KEY_AQUI

📋 Formato y Códigos de Estado

La API utiliza JSON para todas las peticiones y respuestas. Se aplican los métodos HTTP estándar (GET, POST, PUT, DELETE).

✅ Respuestas Exitosas

Código 200 OK

❌ Errores del Cliente

Códigos 400 Bad Request o 422 Unprocessable Entity

🔒 Errores de Autenticación

Código 403 Forbidden

🔍 Recursos No Encontrados

Código 404 Not Found

👥 Clientes (Customers)

GET

Listar Clientes

Devuelve una lista de todos los clientes de un programa de lealtad específico.

Endpoint:

GET /programs/{programId}/customers

Ejemplo de Petición:

GET /programs/4886905521176576/customers

Ejemplo de Respuesta:

[ { "id": 1869, "firstName": "", "lastName": "", "email": "", "phone": "", "points": 0, "humanName": "Anónimo #1869" }, { "id": 1870, "firstName": "Juan", "lastName": "Pérez", "email": "juan.perez@example.com", "phone": "+529994116533", "points": 5, "humanName": "Juan Pérez" } ]
GET

Obtener un Cliente

Devuelve los detalles de un cliente específico.

Endpoint:

GET /programs/{programId}/customers/{customerId}

Ejemplo de Petición:

GET /programs/4886905521176576/customers/6629146321223680
POST

Crear un Cliente

Crea un nuevo cliente en un programa de lealtad.

Endpoint:

POST /programs/{programId}/customers

Cuerpo de la Petición (JSON):

{ "firstName": "Ana", "lastName": "García", "email": "ana.garcia@example.com", "phone": "+525551234567" }
PUT

Actualizar un Cliente

Actualiza los datos de un cliente existente.

Endpoint:

PUT /programs/{programId}/customers/{customerId}

Cuerpo de la Petición (JSON):

{ "firstName": "Ana", "lastName": "García López", "points": 25 }
DELETE

Borrar un Cliente

Elimina permanentemente a un cliente.

Endpoint:

DELETE /programs/{programId}/customers/{customerId}
POST

Añadir Puntos a un Cliente

Añade una cantidad específica de puntos a un cliente.

Endpoint:

POST /programs/{programId}/customers/{customerId}/points/add

Cuerpo de la Petición (JSON):

{ "points": 10 }

🎁 Ofertas (Offers)

GET

Listar Ofertas

Devuelve una lista de todas las ofertas (recompensas) de un programa.

Endpoint:

GET /programs/{programId}/offers
POST

Crear una Oferta

Crea una nueva oferta o recompensa.

Endpoint:

POST /programs/{programId}/offers

Cuerpo de la Petición (JSON):

{ "name": "Café Gratis", "description": "Canjea 10 puntos por un café americano.", "pointsCost": 10, "autoClaim": true }
POST

Actualizar una Oferta

Actualiza una oferta existente.

Endpoint:

POST /programs/{programId}/offers/{offerId}
DELETE

Borrar una Oferta

Elimina una oferta. Nota: Solo se pueden eliminar las ofertas en estado "Borrador" (Draft).

Endpoint:

DELETE /programs/{programId}/offers/{offerId}
POST

Publicar una Oferta

Cambia el estado de una oferta de "Borrador" a "Publicada".

Endpoint:

POST /programs/{programId}/offers/{offerId}/publish
POST

Cancelar una Oferta

Cambia el estado de una oferta "Publicada" para desactivarla.

Endpoint:

POST /programs/{programId}/offers/{offerId}/cancel
